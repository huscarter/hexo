## nod.js

### node安装
```
npm install node-sass -g
```

1. 如果出现错误"EACCES: permission denied, mkdir 使用以下命令安装"，用以下命令
```
sudo npm install node-sass --unsafe-perm --save-dev
```

### node卸载
```
npm uninstall node-sass -g
```
